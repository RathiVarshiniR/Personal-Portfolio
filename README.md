# Full Stack Portfolio Website

A responsive full-stack portfolio website built using React.js, Node.js, Express.js, and MongoDB.

## Features

- Responsive UI
- Project showcase
- MongoDB database integration
- REST API
- Live deployment

## Tech Stack

Frontend:
- React.js
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

## Deployment

Frontend: Vercel
Backend: Render
